# Sample Project Requirements

This document outlines the basic requirements for our sample project.

## Core Feature: User Login

Users must be able to log in to the system using their email and password.
This includes a password recovery mechanism.

## Secondary Feature: Profile Management

Once logged in, users should be able to view and edit their profiles.

### Profile Detail: Avatar Upload

Users can upload a custom avatar image.

## Admin Feature: User Listing

Administrators need a page to view all registered users.
